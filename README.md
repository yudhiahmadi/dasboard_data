# Belajar Membuat Dashboard 

## Setup environment
```
conda create --name dasboard_yh python=3.9
conda activate dasboard_yh
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel
```

## Run steamlit app
```
streamlit run dashboard.py
```
